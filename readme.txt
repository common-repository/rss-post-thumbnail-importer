== Description ==

Import posts from an RSS feed, and set any images included as media enclosures as the featured image.

== Installation ==

1. Upload the `rss-importer-thumbnail` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, Click on RSS & Thumbnails
